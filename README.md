# 🐍 Fundamentos de Programação & Projeto Hamburgueria

Bem-vindo ao repositório de aprendizado de lógica de programação! Este espaço combina conceitos teóricos fundamentais com uma aplicação prática real: um **Sistema de Gerenciamento de Hamburgueria**.

## 💡 Sobre o Repositório

Este projeto é dedicado ao ensino da **Lógica de Programação** para iniciantes, utilizando **Python**. O objetivo é transformar o conhecimento acadêmico em código funcional.

Aqui você aprenderá:
* **Algoritmos:** Sequência lógica para resolução de problemas.
* **Estruturas de Dados:** Uso estratégico de Listas `[]`, Tuplas `()` e Dicionários `{}`.
* **Controle de Fluxo:** Tomada de decisão com `if/else` e repetições com `while/for`.
* **Tratamento de Erros:** Garantir que o programa não pare por entradas inválidas.

---

## 🍔 Projeto de Destaque: Sistema de Hamburgueria

Como aplicação principal dos fundamentos, desenvolvemos um simulador completo de uma hamburgueria via terminal.

### 🚀 Funcionalidades Principais
* **Cardápio Interativo:** Escolha de lanches e combos com atualização de subtotal.
* **Sugestão do Chef:** Uso da biblioteca `random` para indicar um lanche aleatório.
* **Módulo de Delivery:** Cálculo automático de taxa de entrega (Grátis para pedidos > R$60).
* **Trabalhe Conosco:** Sistema de triagem de currículos baseado em idade e experiência.
* **Resumo do Pedido:** Geração de recibo com número de pedido e tempo estimado.
